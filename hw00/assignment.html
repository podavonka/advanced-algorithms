Input

The input is a text file showing the coordinates of Jos�'s stakes in millimeters regarding the map. The first row contains the number of sticks. Each following line comes with coordinates of stakes separated by a space. In fact, the stakes are given in the order in which the fence runs from one stake to the next one, so they link correctly the perimeter of fence.

Output

The output is a single integer number indicating the length of the fence rounded up to meters. Thickness of the stakes is neglected in calculations. The fence is closed, so the program should also add the distance from the last to the first stake.

An example:

Input:

5
20 10
120 10
120 80
100 80
40 60

Output:

1536
